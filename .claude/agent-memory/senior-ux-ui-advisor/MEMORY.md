### Critical Rules (RTL-first)
- Hebrew is the primary language — design and test in Hebrew first
- Never use `text-left`/`text-right` for body copy — use logical properties
- Gold accents signal scarcity and premium, not decoration
- This is a ~30-40M NIS per floor product: every design choice must feel world-class
